# Meu Primeiro Repositório

*Descrição*: Este é o meu primeiro repositório, criado para documentar meu progresso na jornada de desenvolvimento web. O projeto inclui uma página de portfólio pessoal feita com HTML e CSS, com o objetivo de mostrar minhas habilidades iniciais e meu aprendizado contínuo. O site apresenta uma introdução sobre mim, meus interesses e links para redes sociais.

*Objetivo*: Aprimorar meus conhecimentos em HTML e CSS, praticando conceitos de layout, responsividade, e design de interfaces.
